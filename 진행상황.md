## Vercel 배포 체크리스트 (지금 할 일)

### 1) GitHub에 올리기
```bash
git init
git add .
git commit -m "MVP: realtime duo matching"
git remote add origin https://github.com/YOUR_USER/valorant-duo.git
git push -u origin main
```

### 2) Vercel에서 Import
1. [vercel.com](https://vercel.com) → **Add New Project**
2. GitHub repo `valorant-duo` 선택
3. Framework: **Next.js** (자동 감지)
4. **Environment Variables**에 `.env.example` 6개 추가
5. **Deploy** 클릭

### 3) Supabase URL 설정
Supabase → **Authentication → URL Configuration**
- **Site URL:** `https://your-app.vercel.app`
- **Redirect URLs:**
  - `https://your-app.vercel.app/**`
  - `http://localhost:3000/**`

### 4) 배포 확인
```text
https://your-app.vercel.app/api/health  →  {"ok":true}
https://your-app.vercel.app             →  홈 로드
로그인 → Riot 연동 → 매칭 → 보이스/파티/Discord
```
